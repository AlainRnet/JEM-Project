Update check

Tip
Check always for updates. Updating keeps your site secure and mostly provides new useful functionality to you

If you don't visit the home of JEM, Joomlaeventmanager.net, from time to time you should click the button "Updatecheck" to see if a new version of JEM is available.

During the update check a file located at our website is analyzed. The data contained in that file is shown after in a popup including a notice if you are using the latest version of JEM or not.
Keeping JEM up to date is very important. Beside the strongest point, security fixes if needed, updates also often provides new features and fixes for bugs.
